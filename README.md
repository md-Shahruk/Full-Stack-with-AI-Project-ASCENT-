# Ascent

Ascent is a single-project Full-Stack + AI engineering journey built over 27 weeks. Instead of creating dozens of disconnected tutorial projects, this repository continuously evolves one real-world AI-powered SaaS application—from JavaScript fundamentals and TypeScript to React, Node.js, PostgreSQL, Docker, and Retrieval-Augmented Generation (RAG). Every concept learned is integrated into the same codebase, creating a production-ready application while documenting my growth as a software engineer.

---

## Vision

The goal of Ascent is not simply to learn technologies, but to understand **why** they exist and how they work together in a real production system.

By the end of this roadmap, Ascent will become an AI-powered personal learning platform where users can:

- Track learning roadmaps and tasks
- Write and organize study notes
- Upload study materials
- Chat with an AI Mentor using RAG
- Generate AI-powered summaries
- Collaborate in real-time study groups
- Manage personal data securely with authentication
- Experience a complete production-grade SaaS application

This repository also serves as a public record of my engineering journey—from rebuilding my fundamentals to becoming a Full-Stack + AI Software Engineer.
